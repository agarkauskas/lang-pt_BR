Hide Deprecated Applications=Esconder Aplicações Descontinuadas
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Esta definição esconde da lista de todas as apps as aplicações descontinuadas, apps descontinuadas continuarão no entanto a aparecer nas secções de apps instaladas e anteriores
Hide Incompatible Applications=Esconder Aplicações Incompatíveis
This setting will hide applications that are listed as being incompatible with your version of Unraid=Esta definição esconde aplicações que estão listadas como incompatíveis com a versão do Unraid atualmente instalada
Enable Reinstall Default=Ativar Reinstalação como Padrão
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Esta opção ativa um botão adicional nos cartões e pop-ups que permite reinstalar um aplicação usando os valores padrão do modelo Esta opção pode por vezes ser confusa, e é normalmente apenas utilizada quando se pretende instalar a ter a mesma aplicação a correr duas vezes simultaneamente
Enable additional search results from dockerHub=Ativar resultados de procura adicionais a partir do dockerHub
This setting enables CA to retrieve additional search results from dockerHub Note that docker must be enabled for this to be operational=Esta definição permite ao CA ir buscar resultados de procura adicionais a partir do dockerHub Note que o serviço docker tem de estar iniciado para esta opção estar funcional
Allow CA to send any emergency notifications=Permitir ao CA enviar qualquer notificação de emergência
This setting allows CA to periodically scan your system and alert you to the presence of=Esta definição permite ao CA periodicamente pesquisar o sistema e alertar para a presença de
1 Known malware and security vulnerabilities within applications=1 Malware ou vulnerabilidades de segurança conhecidas dentro de aplicações
2 Critical security vulnerabilities within Unraid OS=2 Vulnerabilidades de segurança críticas dentro do Unraid OS
3 Any installed applications which may seriously hinder the operation of your server=3 Qualquer aplicação instalada que possa comprometer seriamente o bom funcionamento do servidor
